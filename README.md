 # Web Development Practice Projects

This repository contains a collection of simple and beginner-friendly *frontend web development projects*. These are built using HTML, CSS, and JavaScript to practice and strengthen web development skills.

## Features

- Basic HTML structure and tags
- CSS styling and layout (Flexbox, Grid)
- JavaScript DOM manipulation
- Event handling and form validation
- Mini projects like calculator, to-do list, and more

## Technologies Used

- *HTML*
- *CSS*
- *JavaScript*
